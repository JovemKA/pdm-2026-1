# Expo-03 - Ant Design Mobile

Projeto de estudo com Expo focado em explorar componentes do UI Kit **Ant Design** para aplicativos mobile.

## Objetivo

Demonstrar o uso de componentes visuais do Ant Design em React Native, incluindo:

- `Button`
- `Card`
- `List`
- `Switch`
- `Badge`
- `NoticeBar`

## Como executar

1. Instale as dependencias:

```bash
npm install
```

1. Inicie o projeto:

```bash
npm run start
```

1. Abra no Android, iOS ou Web:

```bash
npm run android
npm run ios
npm run web
```

## Estrutura principal

- `app/_layout.tsx`: layout raiz com `Provider` do Ant Design.
- `app/index.tsx`: tela de showcase dos componentes.

## Observacao

A biblioteca utilizada foi `@ant-design/react-native`, que corresponde ao ecossistema Ant Design para React Native/Expo.
